# JavaScript Playground (Frontend Practice)

A collection of small interactive features built with vanilla JavaScript.

This project was created as a JavaScript practice playground to demonstrate DOM manipulation, event handling, and basic UI logic.

## Live Demo
https://lynxgva.github.io/JS-mini-projects/

## Included Features
- Color switcher
- Image gallery
- Todo list (add / delete tasks)
- Modal window
- Range slider
- Product filter
- Calculator

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Purpose
This project is part of my frontend portfolio and was built to practice:
- DOM manipulation
- Event handling
- Working with arrays and objects
- Basic UI state management

## Notes
This is a practice playground project. Each feature is implemented independently to demonstrate core JavaScript concepts.
